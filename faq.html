FAQ

よくあるご質問

ご相談・お見積もりは無料です。お気軽にお問い合わせください。

全般について

本当に藝大の現役学生ですか？
はい、東京藝術大学美術学部デザイン科に在籍中です。学生証の確認は購入後ご希望があれば可能です。
なぜこんなに安いのですか？
学生個人での運営により、固定費が圧倒的に低いことと、AIツールを最大限活用することで作業時間を圧縮しています。実績作りのオープン記念価格でご提供中です。実績10件達成後は値上げ予定。
学生さんで大丈夫ですか？
藝大デザイン科で培ったタイポグラフィ・色彩理論・構図の基礎力に加え、Claude / Gemini API での実装力もあります。実例は Works ページ でご確認ください。
作業はどこからされてますか？
主に東京の自宅から。コミュニケーションはココナラ・Lancers のメッセージ機能、または専用フォーム経由でお願いしております。

制作プロセス

どのくらいで納品されますか？
サービスごとに異なります（3〜14日）。各サービスの「お届け日数」をご確認ください。緊急対応はオプション料金で承ります。
修正は何回まで無料ですか？
サービスごとに規定があります。LP制作なら2回まで無料、ロゴなら採用案決定後は無制限。それ以降は1回 +1,500円〜3,000円となります。
途中経過は確認できますか？
中間提案（ワイヤーフレーム or ラフ）を必ず1回お見せします。その段階で方向性を確認していただきます。
気に入らなかったらキャンセルできますか？
ココナラ規約上、納品前のキャンセルは可能です。中間提案後にキャンセルされた場合、内容によっては作業費を一部いただく場合があります（事前にご相談）。

AI機能・実装について

AIを使うので品質が低くないですか？
AIはアイデア出し・量産パートでのみ活用し、最終的な構図・色彩・タイポグラフィは必ず人間が監修します。藝大基準のクオリティを担保します。
AI機能付きLPの API代は誰が払いますか？
OpenAI / Anthropic / Gemini の API キーはお客様にご用意いただきます。1ユーザー1回あたり 0.1〜2円程度。月1000人想定なら月数百〜2000円程度です。最安構成の提案も可能。
プログラミング経験がなくても運用できますか？
はい、納品時に運用マニュアル PDF + Loom 引き継ぎ動画をお渡しするので、プログラミング知識なしでも操作できます。
AIエージェント業務自動化って何ができますか？
毎日のスプレッドシート集計、メール振り分け、Slack 通知、Notion 自動更新、Telegram Bot、AI議事録作成など。
具体例: 私自身が「毎朝7時にニュース要約 + 予定 + カツメッセージを Telegram に届ける個人秘書 Bot」を運用中。

料金・お支払い

支払い方法は？
ココナラ / Lancers のプラットフォーム決済（クレジットカード・銀行振込・コンビニ等）でお支払いいただけます。直案件の場合は別途ご相談。
領収書は発行できますか？
ココナラの領収書発行機能をご利用ください。インボイス対応はまだ未登録ですが、近日中に対応予定です。
追加で別ページの依頼もできますか？
もちろん可能です。リピーター様は5%OFF。HP制作（複数ページ）は別途お見積もり。

著作権・ライセンス

商用利用できますか？
はい、すべてのサービスで商用利用OKです。納品時に著作権譲渡となります。
納品物を SNS で公開してもいいですか？
はい、ご自由にどうぞ。「制作: hanezu design lab」とクレジット表記いただけると嬉しいですが、必須ではありません。
作品を他のクライアントに転売したら？
納品後の二次販売・再配布は禁止しています。ただし、お客様自身が「お客様の事業」で何度も使用するのは制限ありません。
事業主としてあなたの作品を私のポートフォリオに載せていいですか？
「私（クライアント側）の事業のために制作してもらった」として記載いただくのは大歓迎です。

その他

対面打ち合わせはできますか？
基本的にオンライン（テキスト or Zoom）でのコミュニケーションをお願いしておりますが、東京都内の場合のみ対面打ち合わせも可能です（要相談）。
急ぎの案件もお願いできますか？
即日〜1日対応も可能です（オプション料金 +1,500〜3,000円）。事前にご相談ください。
継続契約はできますか？
月額契約も承っております（運用代行 5,000円/月〜、デザイン継続 30,000円/月〜）。詳細はご相談ください。
参考にしてほしいデザインがあるのですが
ぜひお見せください。ヒアリングで「参考デザイン2〜3点」を必ず確認しております。完全コピーではなく、エッセンスを生かした制作になります。
