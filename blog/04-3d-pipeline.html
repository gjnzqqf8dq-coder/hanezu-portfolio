Gemini → Blender → 3Dプリンタ の自動化パイプライン

2026.05.08 · by hanezu · 読了時間 約9分

東京藝術大学デザイン科の現役学生 hanezu です。

「アイデア → 3D実物」までを完全自動化したパイプラインを構築しました。

コンペ応募・プロトタイピング・趣味のフィギュア制作まで、一気通貫で使えます。

全体フロー

[1. 2D画像生成]    [2. 3Dメッシュ変換]   [3. Blender調整]   [4. スライス]      [5. プリント]
Nano Banana Pro → Tripo3D / Hunyuan3D → Blender 5.0 → Creality Print → 3Dプリンタ

Step 1: 2D画像生成

A. Gemini Nano Banana Pro（推奨・無料100枚/日）

  • URL: gemini.google.com
  • 商用利用OK
  • 1枚生成 30〜60秒

3D変換しやすいプロンプトのコツ

例: "a simple ceramic vase with curved silhouette, white background, front view, product photography style, 3D-printable, no text"

避けるべき：

  • 複雑な背景
  • 半透明・反射の多い物体
  • 髪の毛など細かい構造
  • 複数の物体が重なる

Step 2: 2D → 3Dメッシュ変換

🥇 Tripo3D（最高品質）

  • URL: tripo3d.ai
  • 無料枠: 600クレジット（約5回）
  • 出力: GLB / OBJ / FBX / STL

🥈 Hunyuan3D 2.0（完全無料）

  • HuggingFace Spaces で完全無料
  • 待ち時間 1〜3分

🥉 Meshy.ai

  • 無料枠 月10〜20回
  • 高速（30秒〜1分）

Step 3: Blenderで調整

私が公開している自動化スクリプトを使えば、Blender CLI で OBJ → STL を自動変換可能：

/Applications/Blender.app/Contents/MacOS/Blender --background \
  --python blender_import_export.py -- \
  input.obj output.stl 50

ターゲットサイズ（mm）も自動で調整。

Step 4: Creality Print でスライス

Creality Print は CLI でも自動スライス可能：

/Applications/Creality\ Print.app/Contents/MacOS/CrealityPrint \
  --slice 1 --orient 1 --outputdir ./output/ ./model.stl

初回はGUIでプリンタ・フィラメントプロファイルを設定する必要がありますが、その後はCLIから完全自動化可能です。

実例：「kasane / 重ね」procedural design

私が作った3D作品「kasane」は、Blender Python でパラメトリックに生成しています：

# 3層の円盤を重ねて生成
LAYERS = [
    {"radius": 50, "thickness": 8,  "z": 0,  "offset": (0, 0)},
    {"radius": 38, "thickness": 6,  "z": 8,  "offset": (4, 2)},
    {"radius": 28, "thickness": 4,  "z": 14, "offset": (-3, 5)},
]

Python スクリプトで設計するメリット：

  • パラメータ調整が一瞬
  • バリエーション生成が容易
  • 再現性が高い

実物の作品は Works ページ でご覧いただけます。
