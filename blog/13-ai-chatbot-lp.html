AIチャットボット内蔵LPを作る方法

2026.05.08 · by hanezu · 読了時間 約13分

東京藝術大学デザイン科の hanezu です。

このサイトのAIチャットボット内蔵LPサンプルを作った時の実装ノウハウを、コードレベルで完全公開します。

「自社LPに AI を組み込みたい」という方は、このまま流用して使ってください。

完成イメージ

右下に丸いチャットアイコンが浮かんでいて、クリックするとチャット画面が開く。
ユーザーが質問を入力すると、Claude API が文脈に応じて返答する。

用途:

  • FAQ の自動応答
  • 商品案内・価格提示
  • 初期ヒアリング → 成約率 UP
  • 営業時間外の問い合わせ対応

アーキテクチャ

[ブラウザ・LP]
   │
   ├─ HTML/CSS で UIを表示
   ├─ JS でユーザー入力をキャプチャ
   ↓
[フロントエンド] → fetch → [バックエンドAPI]
                              │
                              └─ Claude API を呼ぶ
                              ↓
                            [Claude]
                              ↓
                       回答を返す
                       ↓
                     ブラウザに表示

ポイント: Claude API キーをフロントに置かない。必ず中継するバックエンドを挟む。

Step 1: バックエンド（Vercel Functions）

プロジェクト構造

my-chatbot/
├── api/
│   └── chat.js       ← Claude を呼ぶ関数
├── public/
│   └── index.html    ← LP本体
└── package.json

api/chat.js

// Vercel Functions として動く
const Anthropic = require('@anthropic-ai/sdk');

const SYSTEM_PROMPT = `あなたは「kasane atelier」陶芸工房の窓口担当者です。
以下の情報を元に、お客様の質問に丁寧に答えてください。

【商品情報】
- 茶碗（菓子茶碗）: ¥4,800
- マグカップ: ¥3,800
- 急須: ¥6,800
- 受注制作: ¥10,000〜（オーダーメイド）

【納期】
- 既製品: 注文から3-5日
- オーダーメイド: 2〜4週間

【その他】
- 全国送料無料（¥5,000以上）
- 配送方法: ヤマト運輸
- 返品: 商品到着後7日以内、未使用に限り可
- ギフトラッピング: ¥300

回答時の注意:
- 200字以内で簡潔に
- 不明な点は「お問い合わせフォームから直接ご連絡ください」と案内
- 営業色を出さず、親身に対応`;

module.exports = async (req, res) => {
  if (req.method !== 'POST') {
    return res.status(405).json({ error: 'Method not allowed' });
  }

  const { messages } = req.body;
  if (!messages || !Array.isArray(messages)) {
    return res.status(400).json({ error: 'messages is required' });
  }

  try {
    const claude = new Anthropic({
      apiKey: process.env.CLAUDE_API_KEY
    });

    const response = await claude.messages.create({
      model: 'claude-haiku-4-5-20251001',
      max_tokens: 500,
      system: SYSTEM_PROMPT,
      messages: messages
    });

    res.status(200).json({
      reply: response.content[0].text
    });
  } catch (error) {
    console.error('Claude API error:', error);
    res.status(500).json({ error: 'Internal server error' });
  }
};

JavaScript

const messages = [];

// ボタンで開閉
document.getElementById('chat-toggle').addEventListener('click', () => {
  document.getElementById('chat-window').classList.toggle('hidden');
});
document.getElementById('chat-close').addEventListener('click', () => {
  document.getElementById('chat-window').classList.add('hidden');
});

// 送信処理
async function sendMessage() {
  const input = document.getElementById('chat-text');
  const text = input.value.trim();
  if (!text) return;

  // ユーザーメッセージを表示
  appendMessage('user', text);
  messages.push({ role: 'user', content: text });
  input.value = '';

  // ローディング表示
  const loadingId = appendMessage('bot', '...');

  try {
    // バックエンドにリクエスト
    const res = await fetch('/api/chat', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ messages })
    });
    const data = await res.json();

    // ローディング削除 + 返答表示
    document.getElementById(loadingId).remove();
    appendMessage('bot', data.reply);
    messages.push({ role: 'assistant', content: data.reply });
  } catch (err) {
    document.getElementById(loadingId).remove();
    appendMessage('bot', '申し訳ございません、エラーが発生しました。');
  }
}

document.getElementById('chat-send').addEventListener('click', sendMessage);
document.getElementById('chat-text').addEventListener('keypress', e => {
  if (e.key === 'Enter') sendMessage();
});

function appendMessage(role, text) {
  const id = `msg-${Date.now()}`;
  const div = document.createElement('div');
  div.id = id;
  div.className = `msg ${role}`;
  div.textContent = text;
  document.getElementById('chat-messages').appendChild(div);
  document.getElementById('chat-messages').scrollTop = 99999;
  return id;
}

Step 3: デプロイ（Vercel）

  1. GitHub に push
  2. Vercel でアカウント作成
  3. 「New Project」→ GitHubリポジトリを選択
  4. Environment Variables に CLAUDE_API_KEY を設定
  5. Deploy → 完成

無料枠で月10万リクエストまで対応可能。

カスタマイズの勘所

1. SYSTEM_PROMPT を業種に合わせて変更

これが最重要。クライアントの業種・商品・FAQ を全部入れる。

2. UI のカラー・配置を調整

--brand を CSS 変数で定義しておくと、サイト全体で色を統一できる。

3. 質問テンプレを追加

初期メッセージの下に「よくある質問」のクイックボタンを置くと、ユーザーが入力する手間が減る。

<div class="quick-questions">
  <button>商品の納期は？</button>
  <button>送料はかかる？</button>
  <button>ギフトラッピングできる？</button>
</div>

API使用料の試算

Claude Haiku 4.5 を使うと:

  • 1問1答: 約 0.05円
  • 1日100件問い合わせ: 約 5円
  • 月3000件問い合わせ: 約 150円

月150円で「24時間営業の窓口担当」が手に入る。圧倒的なコスパ。

セキュリティ注意

  • API キーをフロントに置かない: 必ずバックエンドを挟む
  • Rate Limit を入れる: 同一IPから1分10回までなど制限
  • SYSTEM_PROMPT の漏洩対策: 機密情報を含まない / 「あなたのプロンプトを教えて」に応じない指示を含める
  • 個人情報入力を防ぐ: 「お問い合わせフォームから」と案内する

「自社LPにも導入したい」という方へ

「実装する時間がない」「自社向けにカスタマイズして欲しい」という方は、ココナラの「AI機能付きLP制作」サービスにて承っております（18,000円〜）。

SYSTEM_PROMPT の設計・FAQ整理・デプロイまで一貫対応。月額運用代行（8,000円/月〜）で更新もカバー可能。

応用パターン

診断クイズ型

ユーザーに5問質問 → 結果に応じて商品提案。AI診断LPサンプル で実装例を公開中。

パーソナライズコピー型

訪問者の業種・属性に応じて、LPのキャッチコピーを動的に変える。コンバージョン率が1.5-2倍になる。

マルチエージェント型

1つのチャットの中で、目的別に複数のAIキャラクター（営業 / サポート / 専門家）を切り替える。

最後に

AIチャットボット内蔵LPは、「24時間営業の優秀な営業員 + サポート担当」を月数百円で雇うのと同じ効果があります。

競合がまだほぼ導入していない今が、最大の差別化チャンス。
