学生フリーランスの確定申告 完全ガイド（2026年版）

2026.05.08 · by hanezu · 読了時間 約9分

東京藝術大学デザイン科の現役学生 hanezu です。

ココナラ・Lancers などで副業収入のある学生向けに、確定申告について解説します。

※ 本記事は一般情報の共有であり、個別の税務判断は税理士・税務署へご相談ください。

結論：確定申告が必要なライン

原則のライン（学生・アルバイトなし）

副業収入の所得（売上 − 経費）が年間48万円を超えたら、確定申告が必要。

アルバイト併用の場合

アルバイト給与 + 副業所得 の合計で判定:

  • アルバイト給与（年間）+ 副業所得（年間）が合計130万円を超えたら申告必須
  • 親の扶養を維持したい場合は合計103万円以内に収める

第1章：副業収入は「事業所得」or「雑所得」？

「事業所得」の判定

以下の条件をすべて満たせば事業所得として申告可能:

  • ✅ 継続性がある（数ヶ月以上、定期的に活動）
  • ✅ 営利性がある（収益を目的としている）
  • ✅ 主体性がある（自分で営業・契約・納品）
  • ✅ 独立性がある（特定企業の従属関係でない）

ココナラ・Lancers での受託は、これらを満たしやすい。事業所得として扱える可能性が高い。

「雑所得」になるケース

  • ❌ 単発の小遣い稼ぎ（年1〜2件）
  • ❌ 親の手伝いで報酬を得た等

事業所得の方が圧倒的に有利

事業所得として申告できるメリット:

  • ✅ 青色申告控除（最大65万円控除）
  • ✅ 赤字を3年間繰り越せる
  • ✅ 家族への給料を経費にできる
  • ✅ 機材・通信費等を経費計上しやすい

第2章：青色申告のすすめ

青色申告 vs 白色申告

項目青色申告白色申告
控除最大65万円なし
事前申請必要（開業から2ヶ月以内）不要
帳簿複式簿記（やや複雑）簡易簿記
赤字繰越3年間OK不可

青色申告のための事前準備

  1. 開業届を税務署に提出（無料・10分で完了）
  2. 青色申告承認申請書を同時に提出（無料・10分で完了）
  3. 会計ソフト（freee / マネーフォワード）を準備（無料プラン or 月1,000円程度）

所要時間: 30分。控除65万円の効果は年8〜13万円の節税。やらない理由がない。

第3章：経費にできるもの

具体例（ココナラ・LP制作の場合）

  • 💻 パソコン購入費（10万円超は減価償却）
  • 📱 スマホ通信費（事業利用比率分・3〜5割）
  • 🌐 インターネット通信費（同上）
  • 📐 Adobe / Figma 等のサブスク全額
  • 🤖 Claude / Gemini API 利用料全額
  • 📚 書籍・参考書（学習・業務関連）
  • 🚆 交通費（クライアント打ち合わせ等）
  • ☕ カフェ作業費（業務として認められるなら）
  • 📦 郵送費（納品物の郵送等）
  • 💼 家賃の一部（家事按分・1〜3割）

学生だからこそ気をつけるポイント

  • 📚 大学の学費は経費にできない（自己研鑽 = 個人支出）
  • 🎨 画材・課題費は微妙（業務に直接使ったもののみ）
  • 💻 デザインソフトは学生版より商用版が経費としてクリア

第4章：扶養との関係

「年収103万円の壁」と「130万円の壁」

103万円の壁（税法上の扶養）:

  • 合計所得 48万円超 = 親の扶養控除がなくなる
  • 給与所得控除55万 + 基礎控除48万 = 103万円
  • 事業所得の場合、給与所得控除はなく、青色申告控除のみ

130万円の壁（社会保険の扶養）:

  • 収入が130万円超 = 親の社会保険から外れる（自分で国保に加入が必要）
  • 事業所得の場合、収入の判定基準が複雑（売上 vs 所得 で違う）

学生のオススメパターン

パターンA: 扶養維持 + 副業少額（〜年48万円）

  • 確定申告不要
  • 親の扶養維持
  • 気楽だが、収入上限あり

パターンB: 副業で本気（年48万〜130万円）

  • 青色申告で節税最大化
  • 扶養維持しつつ最大化
  • これが私の現在のポジション

パターンC: 完全独立（年130万円超）

  • 扶養から外れる
  • 国保 + 国民年金加入
  • でも、働けば働くほど稼げる

第5章：実際の申告フロー

1月：書類整理

  • 1年分の売上をまとめる（ココナラの「売上明細」をDL）
  • 経費レシート整理
  • 会計ソフトで記帳

2月：確定申告書作成

  • e-Tax または 税務署提出
  • マイナンバーカード必須（事前作成）
  • 所要時間: 慣れれば2-3時間

3月15日：申告期限

第6章：使うべきツール

会計ソフト

  • 🥇 freee（月1,180円〜）：自動仕訳が優秀
  • 🥈 マネーフォワード（月980円〜）：銀行連携が強い
  • 🥉 やよいの青色申告オンライン（年8,000円）：シンプル

レシート管理

  • 📷 スマホでスキャン → クラウド自動保存
  • freee / マネーフォワードのアプリで自動仕訳

銀行口座

  • 事業用口座を分けるのが鉄則
  • 住信SBI / 楽天銀行 が無料で使いやすい

「節税まで含めた副業設計」を依頼したい方へ

ココナラの「Notionテンプレ構築」サービスでは、副業の売上・経費管理を自動化するNotionワークスペースを構築しています（3,000円〜）。

確定申告の準備が「Notion を見るだけで完了」する状態を作ります。

具体例: 藝大式 Notion 運用術

第7章：よくある失敗パターン

失敗1: レシートを捨ててしまう

領収書・レシートは7年間保存義務。捨てると経費計上できなくなる。

対策: スマホで即スキャン → クラウド保存

失敗2: 売上だけ申告して経費を計上しない

「面倒だから売上だけ」は所得税が高くなる。

対策: 経費項目をリスト化 → 月1回の記帳習慣

失敗3: 扶養から外れることに気付かない

年末になって「やばい、扶養から外れちゃう」となる学生多数。

対策: 月1回、年間収入見込みをチェック

最後に

確定申告は「一度やれば来年は半分の時間でできる」性質のもの。

最初は面倒に見えるけど、青色申告控除65万円の効果は強烈。

学生のうちから「税金を理解した経営」をしておくと、卒業後の独立にも直結します。

ぜひ、青色申告 + Notion運用で、計算しやすい副業ライフを作ってください。
