受注に直結するポートフォリオサイトの作り方

2026.05.08 · by hanezu · 読了時間 約7分

東京藝術大学デザイン科の hanezu です。

私は2週間でココナラ7サービス出品 + ポートフォリオサイト11ページを構築し、月10万ペースの副業受託を立ち上げました。

その中で「ポートフォリオサイトの存在」が受注率を 3〜5倍にしたと確信しています。

この記事では、私が実際に作って公開しているポートフォリオサイトの作り方を、完全に再現可能な形で解説します。

私のサイト（参考）: hanezu-portfolio

第1章: なぜポートフォリオサイトが必要か

ココナラ・Lancers だけでは差別化できない

これらのプラットフォームでは：

  • みんなが同じ画像サイズ
  • みんなが同じ説明文構造
  • みんなが同じプロフィール

→ 価格でしか比較されなくなる。

ポートフォリオサイトを別途持つことで：

  • 「このセラーは別格だ」という印象
  • 規約に縛られない自由なアピール
  • ココナラ外からの直案件受注

第2章: 構成（必須5ページ）

  1. Home（トップ）
  2. Works（作品集）
  3. Pricing（料金表）
  4. FAQ（よくある質問）
  5. Contact（お問い合わせ）

これだけで十分。多すぎは逆効果。

第3章: 技術選定

結論：HTML / CSS / JavaScript のみ で OK

WordPress も React も Next.js も必要ない。

理由

  • セキュリティ問題が起きにくい（静的サイト）
  • 表示が圧倒的に速い
  • メンテナンスが楽
  • 無料で公開できる
  • 編集が簡単（テキストエディタで開ける）

推奨ツール

  • VS Code（無料・最強）
  • Figma（無料）：ワイヤーフレーム
  • Vercel / Netlify / GitHub Pages（公開）

第4章: デザインの極意（藝大式）

1. 「書体」が9割

ポートフォリオサイトの印象は書体で決まる。

おすすめフォント：

  • 本文: Inter（無料・モダン）
  • 見出し: Noto Serif JP（和風）or Playfair Display（洋風）
  • モノスペース: JetBrains Mono

→ Google Fonts で全部無料

2. 色は「2色 + ニュートラル」で

推奨パレット例：

Brand: メインの差し色（1色）
Bg: ベージュ or オフホワイト
Text: 限りなく黒に近いダークグレー
Soft: 50%グレー（補助テキスト）
Line: 罫線・区切り（10%グレー）

3. 「余白」が命

藝大デザイン科の真骨頂。

  • 全ての要素に大きめの余白
  • ページ上下: 100px 以上
  • 要素間: 32〜80px
  • line-height: 1.7〜1.8

→「もったいない」と思うくらい余白を取る。

4. アニメーションは控えめに

動きは少ないほどプロっぽい。

第5章: 作品ページの作り方

各作品を「物語」として見せる

  1. ヒーロー画像 + タイトル
  2. メタ情報（年・カテゴリ・使用技術）
  3. コンセプト（200字程度）
  4. 制作プロセス（3-5ステップ）
  5. ギャラリー（3-5枚）
  6. 結果・成果（KPI 等）

「動くサンプル」を入れる

私の場合、ポートフォリオサイト内に：

  • AI診断LP（実際に動く）
  • AIチャットボットLP（実際に動く）
  • 3Dシリーズ（レンダリング画像）

を埋め込んでます。これが圧倒的差別化に。

ポートフォリオ制作を依頼したい方へ

「自分で作る時間がない」「藝大基準のクオリティに仕上げたい」という方は、ココナラのLP制作サービス にてお引き受けしております。

15,000円〜（オープン記念価格）。実例は Works ページ をご覧ください。

第6章: 失敗パターン

失敗1: 「すごい技術」を見せようとする

派手なアニメーション・WebGL・3D表現は不要。クライアントが見たいのは技術力じゃなく「作品の質」。

失敗2: 作品が少なすぎる

3作品以下だと「経験が浅い」と思われる。最低5作品から始めよう（自主制作でOK）。

失敗3: 文章が長すぎる

訪問者は3秒で去る前提。見出しを大きく、段落を短く、箇条書きを多用。

最後に

ポートフォリオサイトは「あなた自身のブランド構築の起点」です。

副業を始めたばかりの今、最初の数百〜数千円の収入を得るより、先にちゃんとしたサイトを作る方が長期的に大きいリターンを生みます。

ぜひ、あなたのスタート地点を作ってください。
