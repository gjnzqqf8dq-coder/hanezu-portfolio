hanezu について

東京藝術大学デザイン科の現役学生が運営する、
「デザイン × AIエージェント実装」の個人スタジオです。

はじめに

こんにちは、hanezu です。
東京藝術大学デザイン科に通う学生で、副業として「デザイン × AIエージェント実装」のクリエイティブスタジオを運営しています。

ここでは、私のバックグラウンド・このスタジオを始めた理由・大切にしている価値観をお伝えします。

始めたきっかけ

藝大に進学して2年目、「画材代・課題費・本代でどんどん出費がかさむ」状況に直面しました。奨学金とアルバイトでは追いつかない。

そこで、「学業と両立できて、自分のスキルを伸ばせる副業」を本気で考えました。

「藝大生らしさを活かしたい。
でも、ただ『学生だから安く』ではなく、
『藝大生だからできること』をやりたい」

たどり着いた答えが、「デザイン × AIエージェント実装」という独自ポジション。
美術系・文系の人が AI 実装を学ぶケースは少なく、技術系の人がデザインを学ぶケースも少ない。
この交差点に、誰も座っていない椅子があった。

2週間でゼロから月10万円へ

ココナラに登録した3日目で初依頼を取り、2週間で:

  • 出品7サービス
  • 受注 12件
  • 売上 12万円（手取り 9万円）

に到達しました。

「特別な才能があったから」ではありません。2週間で月10万円の副業を構築した話 で詳細を公開しています。

Blender Python によるパラメトリックな3D生成。Gemini→Tripo3D→Blender→3Dプリンタの完全自動パイプライン。

3年以上の運用ノウハウ。個人・チーム向けのワークスペース設計から API 連携自動化まで。

業務自動化全般

Webhook、Cron Job、Cloud Functions、GAS。月数百円で動く軽量運用設計が得意。

大切にしている価値観

  • 美しさと実用性の両立 美しさを犠牲にした実用性も、実用性を欠いた美しさも、長続きしません。両方が共存する制作物を目指しています。
  • 透明性のある関係 進捗・修正範囲・追加料金の判断基準。すべて事前に明文化することで、お客様との関係を健全に保ちます。
  • AIを「組み込む」側でいる AIに振り回されるのではなく、AIを業務の中核として組み込む側に立つ。これが2026年以降の決定的な差別化要因だと信じています。
  • 学業との健全な両立 副業はあくまで「学生としての本業」の延長。藝大での学びこそが、この活動の創造性の源泉です。
  • 長期的な関係性 単発受注ではなく、お客様の事業の継続的なパートナーとして関わることを目指しています。月額運用代行は、その表れの一つです。
