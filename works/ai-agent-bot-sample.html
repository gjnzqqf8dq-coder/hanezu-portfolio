毎朝7時のタスク通知 Bot

Notion + Telegram + Claude API · 2023年から運用中

概要

これは私自身が3年前から運用している業務自動化Botです。Notion で管理しているタスクを、毎朝7時に Claude API で整理 → Telegram に通知してくれます。

「忘れる」「優先順位がわからない」を完全に解消し、起床と同時に「今日やるべきこと」が明確になる仕組み。

主な機能

  • Notion からタスク取得: 期限が今日以前 + ステータスが Done でないタスクを自動抽出
  • AI による優先順位付け: Claude が緊急度・重要度を判定し、絵文字で視覚的に表示
  • 各タスクに「ヒント」を追加: 所要時間や着手のコツを Claude が自動生成
  • 定刻配信: Render.com の Cron Job で毎日 JST 07:00 に実行
  • エラー時の通知: 万が一 API エラーが発生しても、Telegram に自動通報

技術スタック

  • Notion API - タスクDB の取得
  • Claude API (Haiku 4.5) - タスクの整理 + ヒント生成
  • Telegram Bot API - 通知配信
  • Render.com Cron Job - 自動実行（無料枠で運用）
  • GitHub Actions - CI/CD

コアコード（一部抜粋）

require('dotenv').config();
const { Client } = require('@notionhq/client');
const Anthropic = require('@anthropic-ai/sdk');
const fetch = require('node-fetch');

const notion = new Client({ auth: process.env.NOTION_TOKEN });
const claude = new Anthropic({ apiKey: process.env.CLAUDE_API_KEY });

async function main() {
  const tasks = await fetchTasks();
  if (tasks.length === 0) {
    return sendTelegram('☀️ 今日のタスクはありません！');
  }

  // Claude でタスクを整理 + ヒント生成
  const enriched = await enrichWithAI(tasks);
  const message = formatMessage(enriched);

  await sendTelegram(message);
}

async function enrichWithAI(tasks) {
  const prompt = `以下のタスクを整理し、各タスクに30字以内のヒントを付けてください。
${tasks.map(t => `- ${t.title} (期限: ${t.due})`).join('\n')}`;

  const res = await claude.messages.create({
    model: 'claude-haiku-4-5-20251001',
    max_tokens: 1000,
    messages: [{ role: 'user', content: prompt }]
  });

  return parseAIResponse(res.content[0].text);
}

main();
